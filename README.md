# TrinityCoreClassic

TrinityCoreClassic is an open-source World of Warcraft sever emulator core that supports a modern vanilla classic client.
Currently supports 1.14.0.40618.

It's a project that was started by https://github.com/Frostshake/TrinityCoreClassic and is based on https://github.com/TrinityCore/TrinityCore (master\wotlk branch).

**Project starting point:** https://github.com/TrinityCore/TrinityCore/tree/wotlk_classic  
**Database starting point:** https://github.com/TrinityCore/TrinityCore/releases/tag/TDB927.22082  
**Vanilla behavior references:** |https://github.com/vmangos/core  
**1.12-1.14.x conversions:** https://github.com/WowLegacyCore/HermesProxy  

# Goal
Creating a server emulator core that gives the true vanilla experience for a modern classic client.

# Current state
You can log in and play for the most part. That being said, there is still a lot of work that still needs to be done, especially regarding the database and class\spell\npc behavior.
